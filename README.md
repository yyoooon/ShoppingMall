# ShoppingMall
**Html, Css, Javascript, Json**  
옷들을 종류, 색상에 별로 볼 수 있도록 필터링 할 수 있는 웹입니다
- **json**에 저장된 데이터를 **fetch API**로 불러와 **js**로 아이템을 만들어주었고
- 아이템에 넣어진 데이터를 기반으로 필터 기능을 구현했습니다
